Overview

Purpose

The purpose of this website is to create a yhatzee dice roller. This will allow an online way of rolling dice for yhatzee, reducing the luggage and space needed for trips and car rides when people want to play yhatzee.

Audience

The intended audience are those that like to play yhatzee, but do not have the luxury of extra space when travelling. This is for the people that like to play yhatzee, but occasionally misplace or lose the dice, or have no way to acces the official yhatzee in that moment.

Dynamic elements

In this website, there will be a button pushed to allow for the dice to roll. Also, there will be a section to ask how many dice the player wants to roll, allowing for one to roll only the needed number of dice. The javascript will display and hide the elements of the dice based on these buttons and fields.

Pseudocode for your project

First, I will set up the basic frame of the website itself, with the header, footer, buttons, and divs to hold the dice. Then, I will create an array holding the information for each type of die that can be displayed by the roller. Then, I will create the function that will select one of the 6 dice to display for each of the 5 slots of dice when the generic roller is pressed. Then, I will add functionality to the number of dice field to change the number of dice that is rolled. Finally, I will add conditions to prevent rolling of 0 dice, or dice greater than 5.
